## Geneviève Bergeron — Guide de référence

## Couleurs principales (codes HEX)

### 🖤 Noir profond
```
#000000 (pur)
#1a1a1a (légèrement atténué)
RGB : 0,0,0 à 26,26,26
```

**Usage :**
- Texte principal (corps de texte, paragraphes)
- Titres H1, H2, H3
- Logo "Octopus."
- Navigation principale
- Éléments de structure

**Ne PAS utiliser pour :**
- Arrière-plans pleins (trop intense)
- Grands blocs de texte sur mobile (préférer #1a1a1a)

---

### 🤍 Blanc cassé / Crème
```
#fafafa (blanc cassé standard)
#f5f5f0 (crème légèrement teinté)
RGB : 250,250,250 et 245,245,240
```

**Usage :**
- Arrière-plan principal des pages
- Zones de respiration visuelle
- Cartes et conteneurs
- Espacement entre sections
- Alternative au blanc pur (#ffffff)

**Pourquoi pas du blanc pur ?**
- Plus doux pour les yeux
- Meilleur contraste avec éléments blancs purs
- Cohérence avec approche minimaliste organique

---

### 🧡 Orange/Corail (COULEUR SIGNATURE)
```
#ff6b35 (orange principal)
#ff8c5a (orange hover/light)
RGB : 255,107,53 et 255,140,90
```

**Usage prioritaire :**
- Boutons d'action principaux (CTA)
- Liens importants
- État hover des liens
- Éléments interactifs clés
- Mise en évidence d'informations critiques
- Icônes d'action

**Exemples concrets :**
- "Réserver une rencontre"
- "Envoyer un message"
- "On se parle →"
- Liens dans le texte (hover)
- Points de navigation actifs

**Règles d'usage :**
- Utiliser avec parcimonie (max 2-3 éléments orange par écran)
- Toujours sur fond clair (blanc, crème, gris très clair)
- Assurer contraste WCAG AA minimum
- Ne JAMAIS utiliser sur fond vert ou gris foncé

---

### 🌿 Vert sauge / Olive
```
#6b7f5b (vert principal)
#8a9a7b (vert clair)
RGB : 107,127,91 et 138,154,123
```

**Usage :**
- Accents pour thématique durable/éco
- CTAs secondaires
- Liens contextuels environnementaux
- Badges/tags liés à l'écoconception
- Icônes durabilité
- Séparateurs thématiques

**Exemples concrets :**
- Tags : "Marketing durable", "Éco-conception", "Core Web Vitals"
- Liens vers ressources environnementales
- Section "Approche durable"
- Icônes plantes/feuilles

**Règles d'usage :**
- Réserver pour contexte durable/environnemental
- Ne PAS surcharger (max 1-2 éléments verts par section)
- Éviter le mélange orange + vert dans le même élément

---

### ⚫ Gris (palette complète)

**Gris foncé**
```
#666666
RGB : 102,102,102
```
Usage : Textes secondaires, sous-titres, métadonnées, dates

**Gris moyen**
```
#999999
RGB : 153,153,153
```
Usage : Textes tertiaires, placeholder, éléments désactivés

**Gris clair**
```
#e0e0e0
RGB : 224,224,224
```
Usage : Séparateurs, bordures fines, lignes de division

**Gris très clair**
```
#f0f0f0
RGB : 240,240,240
```
Usage : Backgrounds de badges/tags, zones de contenu légèrement démarquées

## Hiérarchie visuelle des couleurs

### Niveau 1 — Actions principales
**Orange/Corail (#ff6b35)**
- Boutons CTA primaires
- Liens d'action critiques
- Maximum 1-2 par écran

### Niveau 2 — Actions secondaires / Contexte durable
**Vert sauge (#6b7f5b)**
- Boutons CTA secondaires
- Liens contextuels environnementaux
- Tags durabilité

### Niveau 3 — Structure et contenu
**Noir (#000000, #1a1a1a)**
- Titres
- Corps de texte
- Navigation

### Niveau 4 — Nuances et séparations
**Gris (#666666 à #f0f0f0)**
- Textes secondaires
- Séparateurs
- Backgrounds subtils

## Combinaisons de couleurs

### ✅ Combinaisons recommandées

**CTA principal sur fond clair**
```
Bouton : #ff6b35 (orange)
Texte bouton : #ffffff (blanc)
Background page : #fafafa (crème)
```

**CTA secondaire sur fond clair**
```
Bouton : #6b7f5b (vert)
Texte bouton : #ffffff (blanc)
Background page : #fafafa (crème)
```

**Lien dans texte**
```
Lien : #ff6b35 (orange)
Hover : #ff8c5a (orange clair)
Texte environnant : #000000 (noir)
```

**Badge/Tag**
```
Background : #f0f0f0 (gris très clair)
Texte : #666666 (gris foncé)
Bordure (optionnelle) : #e0e0e0 (gris clair)
```

**Section avec accent durable**
```
Titre : #000000 (noir)
Accent/ligne : #6b7f5b (vert)
Texte : #1a1a1a (noir léger)
Background : #fafafa (crème)
```

### ❌ Combinaisons à éviter

**Mauvais contrastes**
- ❌ Orange sur vert : illisible
- ❌ Gris moyen (#999999) sur blanc : contraste insuffisant pour texte principal
- ❌ Vert sur gris clair (#f0f0f0) : contraste faible
- ❌ Orange sur noir : trop agressif

**Surcharge visuelle**
- ❌ Orange + vert dans le même élément cliquable
- ❌ Plus de 3 couleurs dans une même section
- ❌ Multiples CTAs orange sur le même écran (max 2)

---

## Ratios de couleur recommandés

**Page type :**
- 70% Blanc cassé/crème (espace, respiration)
- 20% Noir/gris (texte, structure)
- 8% Gris clair (séparations, nuances)
- 2% Orange OU Vert (accents, actions)

**Principe clé :** Moins de couleur = plus d'impact quand elle est utilisée.

---

## Accessibilité (WCAG AA)

### Contrastes minimum requis

**Texte normal (16px+)**
- Ratio minimum : 4.5:1
- ✅ Noir (#000000) sur Blanc cassé (#fafafa) : 20.35:1
- ✅ Gris foncé (#666666) sur Blanc cassé : 5.74:1
- ❌ Gris moyen (#999999) sur Blanc cassé : 2.85:1 (insuffisant)

**Texte large (18px+ ou 14px+ gras)**
- Ratio minimum : 3:1
- ✅ Orange (#ff6b35) sur Blanc : 3.35:1
- ✅ Vert (#6b7f5b) sur Blanc : 4.89:1

**Éléments interactifs**
- Ratio minimum : 3:1 avec couleur adjacente
- ✅ Bouton orange sur fond crème : conforme

### Tests de contraste
Utiliser : WebAIM Contrast Checker ou navigateur DevTools

## Cas d'usage spécifiques

### Site web WordPress / Shopify
```css
:root {
  --color-primary: #ff6b35;
  --color-secondary: #6b7f5b;
  --color-text: #1a1a1a;
  --color-text-light: #666666;
  --color-background: #fafafa;
  --color-border: #e0e0e0;
}
```

### Email / Newsletter
- Fond : #fafafa
- Texte : #1a1a1a
- CTA principal : #ff6b35
- Liens : #ff6b35 (avec soulignement)

### Réseaux sociaux (posts)
- Maximum 2 couleurs par visuel
- Orange pour messages d'action
- Vert pour contenu environnemental
- Noir + crème pour contenus informatifs

### Documents imprimés
- CMJN pour Orange : C0 M60 J80 N0
- CMJN pour Vert : C35 M15 J60 N20
- Noir pur : C0 M0 J0 N100

## Checklist couleur

Avant de finaliser un design :
- [ ] Maximum 3 couleurs principales utilisées
- [ ] Orange réservé aux actions primaires (max 2 par écran)
- [ ] Vert utilisé uniquement pour contexte durable
- [ ] Contrastes WCAG AA vérifiés
- [ ] Pas de surcharge visuelle (70% espace blanc/crème)
- [ ] Hiérarchie claire (orange > vert > noir > gris)
- [ ] Cohérence avec identité minimaliste
