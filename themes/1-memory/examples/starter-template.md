## 🎨 Personnalisation

**Comment utiliser ce template** :

1. **Copiez** ce fichier dans `.claude/CLAUDE.md`
2. **Supprimez** les sections non pertinentes
3. **Remplissez** vos choix (supprimez les options non utilisées)
4. **Ajoutez** vos propres règles spécifiques
5. **Adaptez** à votre workflow

### Exemple de Personnalisation

**Avant (template)** :
```markdown
- Framework : [Next.js / React / Vue / Svelte / Angular]
```

**Après (personnalisé)** :
```markdown
- Framework : Next.js 14 (App Router)
```
