# Rédaction Web en Français Canadien

Ce skill couvre les normes et bonnes pratiques de rédaction web adaptées au français canadien (québécois), en tenant compte des spécificités linguistiques, typographiques, légales (Loi 25) et de l'accessibilité.

---

## 🇨🇦 Le Français Québécois et Canadien sur le Web

### Principes fondamentaux

Le français québécois sur le web n'est pas simplement du français européen avec quelques adaptations — c'est une variante vivante avec ses propres normes, son propre registre et sa propre identité.

**Ce que ça implique :**
- Utiliser les recommandations de l'**OQLF (Office québécois de la langue française)** comme référence normative principale
- Adapter les termes techniques et numériques selon la terminologie officielle québécoise
- Respecter les conventions typographiques du français (différentes de l'anglais)
- Tenir compte du registre naturel de la cible québécoise

## ✍️ Principes de Rédaction Web pour le Québec

### 1. Écrire pour le scan, pas pour la lecture linéaire

Les internautes ne lisent pas — ils balayent. Structure qui facilite le repérage :

- **Titres informatifs** (H1, H2, H3) qui se comprennent hors contexte
- **Paragraphes courts** : 3-5 lignes max en ligne, une idée par paragraphe
- **Listes à puces** pour les énumérations de 3 éléments et plus
- **Mots-clés en début de phrase** (structure pyramide inversée)
- **Caractères gras** pour les informations critiques (avec parcimonie)

### 2. Pyramide inversée

Mettre l'information la plus importante EN PREMIER, les détails EN DERNIER.

```
QUOI → POURQUOI → COMMENT → DÉTAILS
(L'essentiel) → (Le contexte) → (Les étapes) → (Les notes)
```

### 3. Lisibilité et niveau de langue

- **Phrases courtes** : 15-20 mots max pour le web
- **Vocabulaire accessible** : éviter le jargon sauf pour les audiences expertes
- **Voix active** de préférence à la voix passive
- **Personnes directes** : "vous pouvez" plutôt que "il est possible de"
- **Éviter les nominalisations** : "choisir" plutôt que "faire le choix de"

### 4. Vous ou tu ?

**En québécois sur le web :**
- **Tutoiement** : marques jeunes, services personnels, contexte décontracté (ex : Octopus, startups)
- **Vouvoiement** : services professionnels, institutions, contexte formel
- **Règle** : choisir un registre et s'y tenir sur tout le site

## 🔤 Typographie Française — Règles essentielles

Les règles typographiques françaises diffèrent de l'anglais et sont souvent mal appliquées sur le web québécois.

### Espaces et ponctuation

| Signe | Règle française | Erreur fréquente |
|-------|-----------------|------------------|
| `:` (deux-points) | Espace insécable avant | Aucun espace |
| `;` (point-virgule) | Espace insécable avant | Aucun espace |
| `?` (point d'interrogation) | Espace insécable avant | Aucun espace |
| `!` (point d'exclamation) | Espace insécable avant | Aucun espace |
| `«` (guillemet ouvrant) | Espace insécable après | Collé au mot |
| `»` (guillemet fermant) | Espace insécable avant | Collé au mot |
| `%` | Espace avant le symbole | Collé au chiffre |
| `$` (dollar canadien) | Symbole après le nombre | Avant le nombre (anglicisme) |

**Exemple :**
- ✅ `Coût : 80 $ / heure`
- ❌ `Coût: $80/heure`

### Majuscules en français

Les majuscules sont **moins fréquentes** en français qu'en anglais.

**PAS de majuscule pour** :
- Jours de la semaine (lundi, mardi...)
- Mois (janvier, février...)
- Langues (français, anglais...)
- Nationalités utilisées comme adjectifs (un client québécois)
- Titres de fonction (la directrice, le ministre)

**Majuscule pour** :
- Noms propres de personnes, d'organismes
- Nationalités utilisées comme noms (les Québécois, les Canadiens)
- Premier mot d'un titre dans une liste

### Guillemets français vs anglais

- **Français** : « guillemets français » (chevrons)
- **Anglais** : "English quotation marks"
- Sur le web québécois : utiliser les guillemets français dans les textes, sauf pour les titres d'œuvres étrangères ou le code

### Tirets

| Tiret | Usage | HTML |
|-------|-------|------|
| `-` (trait d'union) | Mots composés, coupure de mot | - |
| `–` (demi-cadratin) | Incise, plages (2020–2024) | &ndash; |
| `—` (cadratin) | Dialogue, énumération (usage limité) | &mdash; |

**Note web** : le tiret cadratin (—) en cours de phrase est un anglicisme à éviter en français. Utiliser les virgules, les parenthèses ou un point-virgule à la place.

## 🌐 SEO en Français Canadien

### Différences FR-CA vs FR-FR

Certains termes ont des équivalences différentes qui affectent le volume de recherche au Québec :

| Terme FR-FR | Terme FR-CA (plus courant) |
|-------------|---------------------------|
| courriel | courriel (pas "email" — l'OQLF l'a officialisé) |
| portail | portail (identique) |
| logiciel | logiciel (identique) |
| ordinateur portable | laptop (plus courant oralement, mais "ordinateur portable" pour le formel) |
| télécharger | télécharger (identique, pas "downloader") |
| infolettre | infolettre (vs "newsletter" — terme québécois officiel) |
| texto | texto ou message texte (pas "SMS" seul) |
| balado | balado (vs "podcast" — balado est recommandé par l'OQLF) |
| clavardage | clavardage (vs "chat en ligne") |

**Recommandation SEO** : inclure les deux formes quand le volume le justifie (balado/podcast, infolettre/newsletter) — mais prioriser les termes québécois officiels dans les balises meta et les titres.

### Mots-clés locaux

Pour le marché québécois, inclure systématiquement les indicateurs géographiques pertinents :
- "Montréal", "Québec", "Estrie", "Laval" selon la portée géographique
- "au Québec", "en province", "à Montréal" dans le texte
- Nommer les réalités locales (OBNL vs ONG, infolettre vs newsletter, PME québécoises)

## ✊ Écriture Inclusive en Français

### Principes de base (recommandations ULaval + UdeM)

L'écriture inclusive vise à nommer toutes les personnes concernées, sans hiérarchiser les genres. Elle est particulièrement pertinente pour un positionnement progressiste (comme Octopus Marketing).

### Techniques disponibles

**1. Rédaction épicène** (forme non marquée — à privilégier)
- Noms épicènes : "personne", "individu", "membre", "équipe", "clientèle"
- Reformulation : "les gens qui travaillent" plutôt que "les employés"
- Verbes à la place des substantifs : "ceux qui gèrent" → "les personnes qui gèrent"

**2. Doublets complets**
- "les lectrices et lecteurs", "les utilisatrices et utilisateurs"
- Usage : contextes formels, audiences larges

**3. Doublets abrégés (médian)**
- "les lecteur·rice·s", "les utilisateur·rice·s"
- Usage : contextes progressistes, équipes engagées pour l'inclusion
- **Attention** : moins accessible (lecteurs d'écran, dyslexie) — à utiliser avec discernement

**4. Genre neutre**
- "iel", "iels" comme pronom neutre (reconnu dans le Grand Robert depuis 2021)
- Usage : seulement si le contexte le justifie et l'audience le connaît

### Recommandation pour Octopus

Utiliser en priorité :
1. Formes épicènes quand possible
2. Doublets complets pour les audiences formelles
3. Doublets avec point médian dans les contextes militants/progressistes

## 📱 Médias Sociaux en Français (Guide OQLF)

### Règles générales

**Le français s'adapte aux médias sociaux — il ne s'y sacrifie pas.**

- Maintenir un niveau de langue correct même en contexte informel
- Adapter le registre (tutoiement souvent approprié sur les réseaux)
- Les anglicismes courants sont tolérés dans certains contextes (stories, reel, post) mais les équivalents français sont préférables quand ils existent
- Les hashtags : écrire en français (utiliser les deux langues si bilingue)

### Équivalents officiels OQLF pour les médias sociaux

| Terme anglais | Terme recommandé (OQLF) |
|---------------|-------------------------|
| post | publication, billet |
| feed | fil d'actualité |
| story/stories | story (terme entré dans l'usage) |
| reel | reel (terme entré dans l'usage) |
| thread | fil de discussion |
| hashtag | mot-clic (recommandé) ou hashtag |
| like | j'aime, mention J'aime |
| follower | abonné·e |
| influencer | influenceur·euse |
| influencer marketing | marketing d'influence |
| content creator | créateur·trice de contenu |
| community management | gestion de communauté |

## 🔒 Loi 25 — Conformité Langue et Données

**Source** : collectif-web.ca/loi-25-quebec-se-conformer-le-guide/

### Ce que la Loi 25 implique pour la rédaction web

La Loi 25 (Loi modernisant des dispositions législatives en matière de protection des renseignements personnels, Québec) exige :

**Sur le site web :**
- **Politique de confidentialité** rédigée en français (et en anglais si clientèle anglophone)
- **Langue claire** : la politique doit être compréhensible par la personne moyenne
- **Consentement explicite** pour la collecte de données personnelles
- **Bannière de cookies** : si vous collectez des données, vous devez obtenir un consentement éclairé

**Dans les formulaires :**
- Mentionner explicitement les données collectées, leur usage et leur durée de conservation
- Indiquer le droit d'accès, de rectification et de retrait du consentement
- Prévoir un mécanisme pour exercer ces droits

**Dans les infolettres :**
- Consentement explicite avant l'envoi
- Lien de désabonnement fonctionnel
- Identification claire de l'expéditeur
- Information sur la raison de réception de la communication

### Formulations types conformes

**Pour les formulaires de contact :**
> "Les renseignements personnels que vous nous transmettez servent uniquement à répondre à votre demande. Ils ne seront pas partagés avec des tiers. Pour toute question concernant vos données, communiquez avec nous à [email]."

**Pour les formulaires d'infolettre :**
> "En vous inscrivant, vous acceptez de recevoir nos communications. Vous pouvez vous désabonner en tout temps via le lien en bas de chaque courriel."

## 📋 Checklist Rédaction Web Québec

Avant de publier tout contenu web destiné à une audience québécoise :

### Langue et style
- [ ] Français québécois standard (pas d'anglicismes injustifiés)
- [ ] Terminologie vérifiée dans le Grand dictionnaire terminologique ou la Vitrine linguistique
- [ ] Registre cohérent (tu OU vous — pas les deux)
- [ ] Écriture épicène ou inclusive selon le contexte
- [ ] Phrases courtes (15-20 mots max pour le corps de texte)
- [ ] Pyramide inversée appliquée (essentiel en premier)

### Typographie
- [ ] Guillemets français (« ») pour les citations et expressions
- [ ] Espaces avant les signes de ponctuation double ( : ; ? ! )
- [ ] Symbole $ après le chiffre avec espace (80 $, pas $80)
- [ ] Pas de tiret cadratin (—) en cours de phrase (remplacer par virgule ou parenthèse)
- [ ] Majuscules correctes (pas de majuscule pour les jours, mois, langues)

### SEO
- [ ] Termes québécois dans les titres et meta descriptions
- [ ] Indicateurs géographiques si pertinents (Montréal, Québec)
- [ ] Balises structurées (H1 unique, H2-H3 logiques)
- [ ] Alt text des images en français

### Conformité
- [ ] Politique de confidentialité en français (Loi 25)
- [ ] Consentement cookies si applicable
- [ ] Formulaires avec mentions légales sur les données personnelles
